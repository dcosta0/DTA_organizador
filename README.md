# DTA_organizador
Repositório de meu programa em meu modelo DTA e leitor de banco de dados txt ou excel

O programa que permite o usuário modificar arquivos de banco de dados, sendo por txt ou excel, que tenham linhas e colunas. Além disso, ele se baseia num modelo que criei chamado DTA, onde funções D definem valores, T transformam valores e A fazem ações. Além disso, é possível modificar os arquivos e criar gráficos com eles!

# DETALHES IMPORTANTES: 
1) É preciso instalar algumas bibliotecas python no terminal, apenas escreva pip install + o nome da biblioteca

pip install plotly pandas         (em windows)
(As outras bibliotecas já vem com o python)

2) Se você for usar um arquivo txt/csv, dê preferência em separar assim:
coluna1,coluna2,coluna3
valor1linha1,valor2linha1,valor3linha1
valor1linha2,valor2linha2,valor3linha2
valor1linha3,valor2linha3,valor3linha3
etc (com os valores reais, claro, e quantas colunas e linhas quiser)
