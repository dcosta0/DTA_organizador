# DTA_organizador
Repositório de meu programa em meu modelo DTA e leitor de banco de dados txt ou excel

O programa que permite o usuário modificar arquivos de banco de dados, sendo por txt ou excel, que tenham linhas e colunas. Além disso, ele se baseia num modelo que criei chamado DTA, onde funções D definem valores, T transformam valores e A fazem ações.
